To model this problem as a Mixed-Integer Linear Programming (MILP) problem, we need to define variables, constraints, and an objective function that represents the situation described. Let's start by defining the decision variables, objective function, and constraints:

######### Decision Variables
- Let \( x_i \) be the number of times pattern \( i \) is used. These variables are integer-valued since we cannot use a fraction of a cutting pattern.

######### Objective Function
- Minimize the total number of large rolls used, which is the sum of all patterns used:
\[
\text{Minimize} \quad \sum_{i=1}^{N} x_i
\]

######### Constraints
1. **Demand Satisfaction**: For each type of smaller roll \( j \), the total number of rolls produced across all patterns must meet or exceed the demand:
\[
\sum_{i=1}^{N} \text{pattern}_{i,j} \cdot x_i \geq \text{demand}_j \quad \forall j = 1, \ldots, M
\]

2. **Non-negativity and Integer Constraints**: Each \( x_i \) must be a non-negative integer.
\[
x_i \geq 0 \quad \text{and integer} \quad \forall i = 1, \ldots, N
\]

    \item Demands: $\text{demand}_j$ for $j = 1, \ldots, M$
    \item Patterns: $\text{pattern}_{i,j}$ for $i = 1, \ldots, N$ and $j = 1, \ldots, M$
